# Champions Companion — public data & web app

Unofficial fan-made companion tool for Pokémon Champions: matchup cheatsheets and team building.

- **Web app:** served via GitHub Pages from [`docs/`](docs/)
- **Data:** [`data/`](data/) — meta database (current Regulation), moves, abilities, type chart, archetypes, mechanics, roster. Consumed by the mobile app for over-the-air updates. Updated at every meta shift.

Current data: **Regulation M-A** (see `data/regulation.json` for validity and sources).
